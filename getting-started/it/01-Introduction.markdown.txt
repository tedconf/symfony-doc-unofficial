Introduzione
============

Il framework symfony è un framework MVC full-stack che ti aiuta a sviluppare
siti web più velocemente. Inoltre segue una serie di best practice che ti 
aiutano a sviluppare siti web sicuri e facilmente manutenibili. Essere fautore
delle best practice inizia già da quando decidi di installare il framework.

Installare symfony non è molto diverso dall'installare ogni altro software PHP
ma per rendere la tua installazione sicura fin dall'inizio, non puoi limitarti
a caricare tutti i file nella tua web root directory come descritto in molte
guide all'installazione. Il processo richiederà poco tempo in più per installare
symfony, ma lo sforzo ha il suo valore. Inoltre, come per l'installazione di
tutti gli altri software PHP, esistono molte piccole trappole in cui puoi cadere
che possono rendere la tua esperienza più difficile di quanto dovrebbe essere.

Questo tutorial ti spiega tutto quello che dovresti sapere per iniziare un 
nuovo progetto con symfony. Dall'installazione di symfony stesso alla creazione
di un'applicazione alla configurazione del web server; alla fine del tutorial
avrai un'applicazione symfony completamente funzionante, pronta all'uso per il 
tuo prossimo progetto.
